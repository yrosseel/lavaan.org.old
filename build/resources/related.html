Related Projects

On this page, we provide a list of external projects that are related to the lavaan project. The projects are presented in alphabetical order. Please let me know if you would like your project to be added here.

lavaan.survey

A package to take into account survey weights, clustering, strata, and finite sampling corrections in structural equation modeling analyses in R. Wrapper around packages lavaan and survey.

  • Author(s): Daniel Oberski
  • Website: http://wp.daob.org/software/
  • CRAN: lavaan.survey

Onyx

A graphical interface for Structural Equation Modeling.

  • Author(s): Timo von Oertzen, Andreas M. Brandmaier, Siny Tsang
  • Website: http://onyx.brandmaier.de/

semPlot

A package that can be used for unified visualizations of SEM models

  • Author(s): Sacha Epskamp
  • Github: semPlot development
  • CRAN: semPlot

semTools

An R package is collecting useful functions for structural equation modeling in a single, easily accessible place.

  • Author(s): Sunthud Pornprasertmanit, Patrick Miller, Alex Schoemann, Yves Rosseel (and many contributors)
  • Github: semTools wiki
  • CRAN: semTools

simsem

An R package for simulated structural equation modeling. This package can be used to generate data using the structural equation modeling framework. This package is tailored to use those simulated data for various purposes, such as model fit evaluation, power analysis, or missing data handling and planning.

  • Author(s): Sunthud Pornprasertmanit , Patrick Miller, Alex Schoemann
  • Website: http://simsem.org
  • CRAN: simsem
